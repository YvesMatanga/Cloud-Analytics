## Multimodal Image Retrieval

This project demonstrates semantic image search using text-based prompts. Rather than relying only on file names or manually assigned tags, users can describe the image they are looking for in natural language.

### Key Features

- Keyword- and prompt-based image search
- Natural-language query interface
- Semantic matching between text and images
- React-based user interface
- Flask API backend for retrieval logic
